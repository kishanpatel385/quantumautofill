Features

Everything you need, built in

Eight tightly integrated modules, no plugins required.

🌐

Replacements

Per-URL field values auto-fill matching pages on load. Smart key matching reads HTML attributes. Multi-paragraph values into textareas and contenteditables alike.

📋

Text Clips

Reusable snippets inserted via popup, right-click menu, or an in-page overlay (Alt+C). Per-site relevance ranking, pins, recency sort.

⌨️

Text Expansion

Type a shortcode anywhere; expands inline as you type. Case-sensitivity per shortcode, length-priority matching.

🪄

Macros

Record clicks + typing on any page, replay with one click. Selector-resilient wait and scroll-into-view.

✨

Test Data

Realistic sample data (names, emails, phones, addresses) for QA engineers and developers testing their own forms. Generated locally.

🗂️

Form History

Auto-log every submitted form (passwords excluded). Searchable, paginated, auto-pruned to 1000 entries.

🔒

Secure Vault

AES-GCM 256 encrypted local storage with PBKDF2-derived key. Auto-locks on idle, clipboard auto-clears 30s after copy.

💾

Backup & Restore

Single-file JSON export of everything. Vault entries stay encrypted — same master password on the new device decrypts.
